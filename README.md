# 🚀 My Project Roadmap Dashboard

Un dashboard minimalista y eficiente diseñado para gestionar el progreso de múltiples proyectos de forma independiente. Construido con **React** y **Vite**, este panel te permite visualizar tus metas de un vistazo y mantener un registro detallado de cada paso.

---

## ✨ Características Principales

* **Persistencia Local:** Olvídate de perder datos. Todo se guarda automáticamente en el `localStorage` de tu navegador.
* **Gestión Multi-proyecto:** Crea tantos proyectos como necesites, cada uno con su propio flujo de trabajo.
* **Visualización de Progreso:** Barras de progreso dinámicas para ver cuánto te falta para completar cada meta.
* **Roadmaps Independientes:** Entra en cada proyecto para añadir, editar o marcar tareas de forma aislada.
* **Interfaz Rápida:** Gracias a Vite, la experiencia de usuario es instantánea y fluida.

---

## 🛠️ Stack Tecnológico

* **Framework:** [React](https://reactjs.org/)
* **Herramienta de Construcción:** [Vite](https://vitejs.dev/)
* **Almacenamiento:** Browser LocalStorage API

---

## 🚀 Instalación y Uso

1. **Clona el repositorio:**
```
   git clone [https://github.com/tu-usuario/nombre-del-repo.git](https://github.com/tu-usuario/nombre-del-repo.git)

```

2. **Instala las dependencias:**
```
npm install

```


3. **Lanza el servidor de desarrollo:**
```
npm run dev

```


4. **Abre tu navegador** en `http://localhost:5173`.

---

## 📝 Cómo funciona

1. **Añadir Proyecto:** Usa el botón principal para darle nombre a tu nueva aventura.
2. **Configurar Roadmap:** Dentro de cada proyecto, define las tareas o hitos necesarios.
3. **Seguimiento:** A medida que marques los *checks*, verás cómo la barra de progreso general se actualiza en el dashboard principal.
